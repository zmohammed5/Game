# A Garden For You

A browser-based exploration game built as a romantic gift for a significant other. Built with vanilla HTML5, CSS3, and JavaScript — no frameworks or dependencies.

## Overview

An atmospheric gothic-romance themed exploration game where players navigate through interconnected environments, collecting hidden notes that reveal a secret message. Features original narrative writing, dynamic soundtrack, and multiple visual effects.

## Features

**Gameplay**
- 4 explorable areas with unique atmospheres (Garden, Clearing, Greenhouse, Cemetery)
- 12 collectible story notes with hidden letter fragments
- Secret message reveal system
- Progress saving via localStorage
- Replayable experience

**Technical**
- Pure vanilla JavaScript — no libraries or frameworks
- HTML5 Canvas rendering with layered visual effects
- CSS3 animations for UI transitions
- Responsive audio system with area-specific soundtrack
- Collision detection and movement physics
- Particle systems (fireflies, floating petals, mist effects)

**Visual Effects**
- Animated starfield background
- Dynamic fog and mist layers
- Glowing firefly particles
- Floating petal effects
- Vignette overlay system

**Audio**
- 6-track curated soundtrack
- Area-based music transitions
- Volume control and track skipping

## How to Run

1. Clone or download the repository
2. Open `For You, My Angel.html` in any modern browser
3. No server required — runs entirely client-side
